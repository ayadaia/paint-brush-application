           Paint Brush Application

# Project Overview:
Paint Brush is a Java-based application using JFrame that allows the user to draw basic shapes with different colors and line styles.  
The application also supports free hand drawing, erasing, clearing the drawing area, and undo operations.
---

# Technologies Used:
- Java
- Swing (JFrame, JPanel)
- AWT (Graphics, Mouse Events)
- Object oriented programming 
---

# Features:

Color Selection
Users can choose a color before drawing:
- Red
- Green
- Blue

Shape Drawing
Users can draw shapes by dragging the mouse:
- Rectangle
- Oval
- Line

Free Hand Drawing
- Allows the user to doodle freely by dragging the mouse.

Eraser Tool
- Enables erasing specific parts of the drawing area.

Clear All
- Clears all drawings from the drawing area with a single click.

Dotted Stroke
- When checked, shapes are drawn with dotted lines.
- When unchecked, shapes are drawn with solid lines.

Filled Shapes
- When checked, Rectangle and Oval shapes are filled with the selected color.
- When unchecked, shapes are drawn as outlines only.

---

▶️ How to Run the Project
1. Open the project in NetBeans or any Java IDE.
2. Run the main JFrame class(MyPaintBrush.java)
3.  Use the buttons to start drawing
